## Features

- **Slidev**: Powerful presentation tool for developers.
- **Firebase Hosting**: Fast and secure hosting for the presentation.
- **Gemini 3 Flash & Nano Banana**: AI-powered content generation.
- **Genkit**: Developer-friendly AI integration.

## Getting Started

### Local Development

To start the development server:

```bash
npm run dev
```

### Build

To build the project:

```bash
npm run build
```

### Deployment

To deploy the presentation to Firebase Hosting:

```bash
npm run deploy
```

This command runs `npm run build` and then `firebase deploy --only hosting`.
